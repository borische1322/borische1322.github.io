2D top-down ARPG Souls-like Game


School project ––– from scratch

Inspired by many ARPG games and recent VR trend, my teammates and I have decided to combine ARPG and VR together to create a game that is rich in story telling, combat and immersion.



Main features:
1. remote-free hand-controlled system
2. Open world
3. Enemy AI using Reinforcement Learning
4. Rich Storyline



My Responsibility:
1. Writing the Storyline
2. construct the landscape of the world in Unity3D
3. Enemy AI using RL with a unity package called MLAgent.



Details can't be published yet as it is still on going and is a school Project
